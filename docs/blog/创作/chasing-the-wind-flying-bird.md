# 开始，追风飞鸟

**「终会遗忘，终将消逝」**

刚又被班主任提醒作业未交，其实她也知道我根本就没写。无感地放学，准备放首歌在路上，却看见家人的车。  
上车，车并不是往家里走，我们突然被告知要前往学校一趟，父母很激动，我却还没怎么缓过神来。  
尚未认真思考一番，我便出现在了学校二楼，老师宣讲，然后让我们签字，上交，一切如梦幻般，仿佛不真实。  
离开学校已是晚十一点，车窗外灯火零星，我们没有说话，没有力气，也没有遗憾。

那是甲辰年的深秋，夜晚一片静谧，我却睡不着。我问身旁的赶月姐，广义的说，我是不是考上高中了？  
她笑笑，说是的，这一切都是我的努力取得的。她抱住我，问我还想继续努力吗？  
我在她怀里安适地合上双眼，欸，如果赶月姐在的话，不想努力了。  
许久未有言语，当我快要睡着时，赶月姐放开了我，又牵住我的双手，于是开口。  
「唉呜，今后我肯定是不在了，不过，你已经走上幸福的道路了呢，我放心。」  
「啊啊，怎么会……要是没有赶月姐的话，我。」  
「嗐……你别难受，呃，你可以自己做自己的赶月姐姐啊，喵？」  
「啊，我自己吗，呜。」  
望了望自己的身体，一阵寒意，一阵恶意，我不禁缩起全身，又靠进姐姐的怀里。  
「姐姐不要离开我呜呜。」  
「诶，陪你到毕业吧。」

中考很忙，让我暂时忘记了高中的一切，甚至失去了食欲，爱欲。慢慢地，我变得成熟却麻木。  
毕业那天和女朋友柳颖在 KTV 唱完，心里莫名有些空虚，想和赶月分享些许美好，听她扑哧几声笑，可终于发现，赶月姐真的不在了。  
我尝试接受，于是发完朋友圈后，无感地躺下。  
弹出消息提示，在床上又打开，准备和柳颖约好明天的聚会，却发现已被拉黑。  
红点弹出的瞬间，我内心仍无波澜，而当发现所有联系方式都被删时，我开始急躁不安。  
游戏好友呢，我尝试链接，却亲眼看见她将好友删去，破防，不足以形容我的感受。

那晚我是一直哭着的

后来在对面学校的录取名单上看见柳颖的时候，虽不清详情，我仍选择先放下感情，却发现我已经做不了自己的姐姐了。  
当列表空无消息，我应如何自己安慰自己？

没有人真正喜欢我，我当时第一次这样想。

堕落，只在一瞬间。

然而肌肉记忆打断了施法，顺手点开的音乐，描绘着绝境中的深情，脑海里似有大剑挥舞。我双手微颤，又看向了自己的身体

还是肮脏，不忍直视。  
目光于是移远，顿觉生命馈赠之时间不应浪费。

**「走那条全新的路，可以吗」**

大致一个月后，身体情况基本稳定，在晚上有一次感动幸福快乐时，我正整理初中的资料，发现自己原为初中写过一首歌，于是补完，试着谱上曲子，清唱，是岁月的味道。

:::info[原题：等风来不如追风去]{open}
又到春风挂梢，夕阳满霞；我们等风不来，逐梦浪花；  
曾见银装素裹，茂物纷华；不悔青春有梦，一起出发：  
我们追风去吧，追风在那盛夏，六七月的芳花，总有梦想发芽；  
我们追风去啦，追风在这盛夏，看无人的角落，还有一束鲜花。

老师世上有没有办法，让我可以永远不长大；  
我向往人间如梦的繁华，却害怕往前走越过悬崖。  
看望盎盛远方，总有灯光；夜晚灯火绚烂，灰烬辉煌；  
也许你会跌倒，却坚信信仰；憾叹暮色渐晚，也应飞翔。

年轻的长叹变成风推着我俯瞰，踩过的泥潭早已花开五彩斑斓。  
当有一天你不再纠结于答案，当我们重逢于天涯或沧海，  
这一路的辛酸和孤单和愤慨，要一吐为快。

我一定会到达，想去的地方；旅途结束后，也许有新渴望。  
我们追风去吧，追风在那盛夏，追风赶月不停留，飞鸟如是述说；  
我们追风去啦，追风在这盛夏，再见你好这浪涛，祝福所有美好。

我们追风去啦，随着飞鸟的话，在春山的尽头，绽放青春之花。  
我们追风去吧，回见你我的家，不悔潮首的宏愿，试要真我无暇。  
我们追风去吧，越过梦想出发，漂泊在这片蔚蓝，世界为你敞开。  
我们追风去啦，约好在这盛夏，群星之下的人啊，愿你乘风去吧。
:::

在作好曲子的时候，我在笑。

我将原本写在上面的名字擦去，改为曲名。  
它叫追风飞鸟说。

而又在最下方，轻盈地，自然地画下署名  
我叫赶月。
